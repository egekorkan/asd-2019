Submission Instructions

Submitted papers must be neither previously published nor under review by another workshop, conference or journal.

File format: PDF

Authors of accepted papers must submit the LaTeX source code for their camera-ready papers (this is required by OASIcs).

Maximum number of pages: 15

Single blind review process: manuscripts can include the author names and affiliations.

Format: OASIcs has a specific style of papers format. Submissions must therefore follow the OASIcs formatting instructions.

Submission Site : https://www.softconf.com/date19/ASD2019/
